## ✨ 功能

### 基础版（免费）

- 📑 无限导入/导出书签
- 📁 书签管理
- 🎨 基本主题定制
- 🔍 书签搜索

> 注：MagicNav 基于 [Pintree](https://github.com/Pintree-io/pintree)（MIT许可证）开发。

### 改进部分

- **布局和UI优化**：统一了部分按钮样式，修复了尺寸不一致和扁平按钮问题，确保视觉一致性
- **移动端友好的侧边栏**：在移动端点击侧边栏菜单项后，侧边栏自动收起
- **移动端按钮优化**：后台页面在移动端仅显示按钮图标，隐藏文本，优化空间利用
- **优化的Footer对齐**：解决了Footer元素对齐问题，提升了页面底部美观度
- **增加社交媒体url**：Pinterest、Behance、Dribbble等，微信公众号支持添加二维码图片。
- **版权信息支持超链接**：在导航网站底部显示版权信息，支持<a>标签添加超链接。适合国内用户添加备案号链接。
- **还有一些小功能和优化**：自己部署后去探索，作为个人使用基本足够了。


## 🛠️ 技术栈

- **部署**: Vercel
- **数据库**: PostgreSQL
